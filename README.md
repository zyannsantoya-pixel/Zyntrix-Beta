# Zyntrix - Frontend Gadget Store

This is a frontend-only e-commerce website for "Zyntrix", built with vanilla HTML, CSS, and JavaScript.

## Project Structure

- **index.html**: Home page.
- **products.html**: Product catalog.
- **about.html**: About Us page.
- **contact.html**: Contact page with Google Map.
- **style.css**: Global stylesheet.
- **script.js**: Global logic (Cart, Navigation, Modals).

## Features

- **Multi-Page Layout**: Separate HTML files for each section.
- **Dynamic Cart**: Cart functionality persists across pages via `localStorage`.
- **Responsive Design**: Mobile-friendly navigation and grid.
- **Location Map**: Embedded Google Map showing Cebu City.
- **User Authentication**: Simulated Login/Register with `localStorage` persistence and redirection.

## How to Run

Simply open `index.html` in any web browser.
